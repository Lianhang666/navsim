# NavSim

NavSim 是一个导航模拟器，用于测试和评估自动驾驶和导航算法。

## 项目结构

- `agents/`: 包含各种智能体实现
- `common/`: 通用工具和函数
- `evaluate/`: 评估工具和指标
- `planning/`: 路径规划算法
- `visualization/`: 可视化工具
